# Code of Conduct

We are committed to making the DIGIT community welcoming, safe, and inclusive for everyone.  

## Expectations
- Be respectful and considerate in all interactions.  
- Use inclusive language and be open to different viewpoints.  
- Focus on constructive collaboration and helping others.  
- Report security issues responsibly and respect project licenses.  

## Unacceptable Behavior
- Harassment, discrimination, or personal attacks.  
- Trolling, insults, or sustained disruption of discussions.  
- Sharing private information without consent.  
- Introducing plagiarism, license violations, or security risks.  

## Enforcement
Maintainers may take action on unacceptable behavior, including warnings or removal from the community.  

If you experience or witness unacceptable behavior, please report it to:  
**security@digit.org** (for security)  
**contact@egovernments.org**
